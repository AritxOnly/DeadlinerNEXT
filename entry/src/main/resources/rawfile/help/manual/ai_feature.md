# Deadliner AI

Deadliner AI 是您的智能任务助理。它不仅能听懂简短的指令，更能处理复杂的长文本，帮您从杂乱的信息中瞬间整理出待办清单。

## 1. 核心功能

### 批量智能提取
您不需要按照特定格式输入，也不必一个个创建任务。
直接把**大段文字**（如会议纪要、聊天记录、邮件正文）复制粘贴给 AI，它会自动分析并提取出其中包含的**一个或多个**任务。

- **场景示例**：
  粘贴一段工作群的聊天记录：
  > “@小王 这周五前把月度报表发给我。另外下周一我们要开全员大会，@小李 记得提前订好会议室，并通知大家准备 PPT。”

  **AI 将自动识别并生成 2 个任务**：
    1. **提交月度报表** (截止：本周五)
    2. **预定会议室并通知全员** (截止：下周一)

### 精准要素识别
AI 能够精准识别以下要素：
- **任务名称**：要做什么。
- **时间信息**：具体的截止时间（支持“下周三”、“明天下午”等自然语言）。
- **备注细节**：保留任务的上下文信息。

## 2. 如何使用

1. 点击主页 **右上角** 的 **魔法棒图标** (✨)，打开 AI 面板。
2. 在底部的输入框中输入或粘贴您的计划/文本。
3. 点击发送。
4. AI 会回复一个包含**所有识别结果**的任务卡片列表。
5. 点击对应卡片上的 **“去创建”** 按钮，即可将任务保存到您的清单中。

## 3. 配置与隐私

### 模型配置
Deadliner AI 需要连接大模型服务才能工作。
在使用前，请先前往 <AI 配置>(settingsAI) 页面，根据 <配置指引>[ai_config]，填入您的 API Key 并选择服务商（如 DeepSeek）。

### 隐私安全
- **数据流向**：您的对话数据仅用于当前的任务提取请求，直接发送至您配置的模型服务商。
- **不存储**：Deadliner **不会** 在服务器端存储您的任何聊天记录或任务数据。
- **透明性**：所有 AI 生成的内容都会标记“AI生成内容，请核查”，最终的确认权永远在您手中。
