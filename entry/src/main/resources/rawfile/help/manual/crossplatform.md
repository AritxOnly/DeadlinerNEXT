# 跨平台体验

Deadliner 不仅仅是一个单机应用。我们致力于构建一个全平台的原生任务管理生态，无论你是在通勤途中、办公桌前，还是专注于鸿蒙原生体验，都能“随时随地，把事做完”。

## 多端适配

为了追求极致的性能与符合直觉的交互体验，Deadliner 在三个平台上均采用适合该平台的技术栈分别维护，但保持了设计语言与核心体验的高度一致：

### HarmonyOS NEXT

基于 **ArkTS** 与 **ArkUI** 原生构建，带给你纯净、流畅的鸿蒙体验。

- **系统级卡片**：支持 <小组件>[widgets] 放置于桌面或锁屏，让进度清晰可见。
- **AI 深度集成**：接入系统级或大模型能力，通过 <AI 任务助手>[ai_feature] 智能规划。

### Android

采用最新的 **Material 3 Expressive** 设计语言，兼顾美观与效率。

- **丰富的小组件**：提供桌面小组件、临近通知推送与每日概览推送。
- **概览统计**：支持通过图表回顾每月的任务完成轨迹。
- **AI 任务助手**：同样具备 AI 能力，支持自然语言描述任务并自动提取。
- **流体云适配**：支持实时通知与流体云（Live Activities），让倒计时更加直观。

### PC 客户端 (Beta)

基于 **React + Electron** 构建，延续了 **Material 3** 的设计语言，并专为“键鼠操作”优化。

- **键盘驱动**：支持全局快捷键与命令面板，双手不离键盘即可完成创建、完成与归档。
- **批量管理**：支持多选、批量改期、标签管理与拖拽排序，适合进行每周复盘或重度任务整理。
- **更强回顾**：提供周视图、月视图及更详细的统计视图。

## 数据同步

Deadliner 坚持**“数据自主”**与**“无账号体系”**的理念。我们通过标准协议打通生态，将数据的所有权交还给你。

通过配置 **WebDAV**，你可以让上述所有平台的 Deadliner 保持数据实时同步。

- **隐私安全**：数据存储在你信任的网盘（如坚果云、Nextcloud、Alist 挂载的 NAS 等），而非我们的服务器。
- **无缝互联**：在 PC 端批量整理的任务，秒级同步至 HarmonyOS 和 Android 设备。

👉 **立即开启同步**：在使用跨平台功能前，请先进行 <WebDAV 同步配置>[webdav_config]
