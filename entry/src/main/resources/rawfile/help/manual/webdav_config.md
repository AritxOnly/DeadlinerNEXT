# 什么是 WebDAV？
简单来说，它就像一个 **“云端保险箱”**。开启后，您的任务数据会自动备份到云盘。即使更换手机，只要登录同一个账号，数据就能立刻回来。

## 第一步：开启功能
进入 **我的**，找到 <数据同步与备份>(settingsWebDav) 卡片。
点击顶部的开关，将其设为 **开启** 状态。

![开启开关](help/images/webdav_switch.webp)

## 第二步：注册云服务账号
下面以**坚果云**为例演示注册云服务的方法。

进入坚果云官网 https://www.jianguoyun.com/ ，点击右上角注册一个账号。登录后进入个人中心。
右上角点击自己的用户名，点击账户信息进入配置界面。

![账户信息入口](help/images/jianguoyun_account_info.webp)

进入安全选项，在第三方应用管理里点击“添加应用”。

![安全选项](help/images/jianguoyun_security_options.webp)

按照提示添加应用名称，复制专用密码即可进入下面的设置。

## 第三步：填写账号信息
开启后，您需要填写以下三项信息（以**坚果云**为例）：

**1. 服务器地址**
  您的 WebDAV 服务器地址。具体请查看您的云提供商官网上标注的地址。
  *注意：地址必须以 https 开头。*

**2. 用户名**
  您注册的 WebDAV 账号。

**3. 密码**
  您的 WebDAV 账号的密码。

填写完成后，点击底部的 **保存并与云端同步** 按钮。如果提示 **“同步成功！”**，说明配置成功！

## 如何进行多端同步？
Deadliner 现已完美支持 **鸿蒙 (HarmonyOS)**、**安卓 (Android)** 以及 **PC 端**（Windows、macOS 与 Linux）。无论您是在通勤路上使用手机，还是在办公室使用电脑，任务列表都能随身而动。

**操作步骤：**

1. **下载应用**
   在您的平板、电脑或备用手机上安装 Deadliner。

2. **配置 WebDAV**
   进入新设备的设置页面，填写与当前设备 **完全一致** 的：
    - 服务器地址
    - 用户名
    - 密码

3. **自动拉取**
   点击 **保存**。系统会自动识别云端的任务备份，并将其同步到当前设备。

> **💡这也是数据迁移的方法**
> 如果您更换了新手机，只需按照上述步骤登录 WebDAV，所有数据就会自动恢复，无需手动导数据。

**防冲突小贴士：**
为了确保数据准确，建议在切换设备使用前（例如从手机换到电脑），先在当前设备上**等待自动同步完成**或**手动点击同步**，确保云端数据是最新的。
