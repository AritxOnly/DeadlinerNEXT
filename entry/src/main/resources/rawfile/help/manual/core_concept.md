# 重塑任务管理体验
Deadliner (待办日历) 不仅仅是一个任务清单，更是一种**让任务可感，让进度可见**的生活方式。

我们拒绝繁杂的非必要功能，一切只为 **“把事做完”** 而设计。

## 核心设计理念

### 1. 原生与极简
无论是 HarmonyOS 的 **ArkUI** 还是 Android 的 **Material 3**，我们坚持使用最原生的设计语言。
- **纯净体验**：无广告、无开屏、无多余干扰。
- **直观交互**：滑动、点击、长按，将常用操作浓缩在三步以内，降低学习成本。

### 2. 数据完全自主
我们采用了独特的 **“自备后端模式”**。
- **无强制账号**：所有信息默认在设备端侧存储，您的隐私只属于您。
- **WebDAV 同步**：通过标准的 WebDAV 协议（支持坚果云、NAS 等），打通鸿蒙、安卓与 PC 平台，让数据无缝流转。

### 3. 智能与可视化
- **AI 任务拆解**：接入大模型服务，从长段文本中智能筛选任务信息，一键生成待办。
- **概览统计**：通过图表清晰直观地展示任务趋势，每月有总结，月月看得见。
- **桌面卡片**：无需打开应用，只需一眼即可掌握任务进度。

## 为什么选择我们？
因为我们相信，好的工具应该像空气一样：**在您需要时触手可及，在您专注时安静隐形。**

我们坚持开源开放（GPL-3.0 协议），致力于构建一个跨平台、高效率、可信赖的任务管理生态。
