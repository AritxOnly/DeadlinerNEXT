# 如何添加任务

在 Deadliner 中，创建一个新任务非常简单。你可以选择手动填写详细信息，或者利用 AI 助手一键生成。

## 1. 进入创建页面

在应用主页，确保顶部的标签栏选中了 **“任务”**（默认状态）。

点击屏幕底部的 **“+” 浮动按钮**。

> **提示**：根据您的设置，添加页面可能会以 **半屏弹窗 (Sheet)** 或 **全屏页面** 的形式弹出，但操作方式是完全一致的。

## 2. 填写任务信息

进入创建页面后，您可以完善以下信息：

- **任务名称**（必填）：简要描述你要做什么，例如“完成微积分作业”。
- **开始时间**：任务预计开始的时间。
- **结束时间 (Deadline)**：任务的截止日期。Deadliner 会根据这个时间计算倒计时。
- **备注**（可选）：您可以填写更多细节，例如作业的具体页码或参考资料链接。

## 3. 使用 AI 智能录入（推荐）

不想手动逐个选择时间？试试顶部的 **AI 输入框**。

只需在带有 ✨图标的输入框中像聊天一样输入自然语言，例如：

> “下周五下午3点交英语报告”

点击发送或确认，AI 会自动识别：

- **任务名称**：交英语报告
- **结束时间**：下周五 15:00

识别完成后，您可以检查并微调信息，然后保存。

## 4. 保存任务

确认信息无误后，点击右上角的 **“对勾 (√)”** 按钮即可保存。

保存成功后，新任务将立即出现在您的首页列表中，倒计时随即开始！

# 如何添加习惯

除了单次的任务，Deadliner 还支持创建长期坚持的习惯。无论是“每天喝八杯水”还是“今年读完50本书”，都可以在这里设定。

## 1. 切换到习惯页面

在应用主页，请先点击顶部的标签栏切换到 **“习惯”**。

点击屏幕底部的 **“+” 浮动按钮** 进入习惯创建页。

## 2. 设定习惯规则

习惯的配置比任务稍丰富一些，您需要设定它的**周期**与**目标**：

- **习惯名称**：例如“晨跑”或“背单词”。
- **重复周期**：选择习惯的执行频率。
^ **每日**：每天都要打卡。
^ **每周**：按周结算进度。
^ **每月**：按月结算进度。
- **目标设置**：
^ **坚持模式**（默认）：只需设定**每次/频次**。例如“每日”周期下填“3”，意味着每天要完成 3 次打卡才算达标。
^ **定量模式**：设定一个宏大的**总目标数量**。例如“今年要跑完 1000 公里”，系统会追踪您的累计进度，直到达成总数。

## 3. AI 帮你定计划

同样，您可以使用顶部的 **AI 输入框** 快速创建习惯。

在输入框中输入您的计划，例如：

> “我想要每天背20个单词”
> “计划每周去健身房3次”

AI 会自动识别出：
- **名称**：背单词 / 健身房
- **周期**：每日 / 每周
- **频次**：20次 / 3次

## 4. 开始坚持

确认设置无误后，点击右上角的 **“对勾 (√)”** 保存。

习惯将以进度条的形式展示在列表中。每完成一次，点击卡片，进度就会向前推进一步。
