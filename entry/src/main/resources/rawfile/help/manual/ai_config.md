# 什么是 Deadliner AI？
Deadliner 鸿蒙版内置了强大的 AI 辅助功能。它可以帮您：
- 从一段复杂的文字中（如微信聊天记录、邮件通知）自动提取任务信息。
- 自动规划任务的开始时间和截止日期。
- 将原本繁琐的录入过程变成“一键生成”。

**特别说明：实名认证与合规**
根据国家《生成式人工智能服务管理暂行办法》及相关法律法规要求，使用生成式 AI 服务需要进行实名身份认证。

为保障合规与安全，Deadliner 鸿蒙版采用以下认证方式：
1. **华为账号登录**：您需要登录华为账号，作为实名认证的基础凭证。
2. **API Key 校验**：您填写的 API Key 将经过有效性校验，确保服务可用且归属于您本人。

## 第一步：获取 DeepSeek API Key

Deadliner 目前仅支持 **DeepSeek (深度求索)** 模型。这是一家国产顶尖的大模型厂商，价格实惠且性能强大。

**1. 注册与登录**
使用浏览器访问 [DeepSeek 开放平台](https://platform.deepseek.com)，注册或登录您的账号。

![登录DeepSeek](help/images/deepseek_platform.webp)

**2. 充值额度**
DeepSeek API 是付费服务（但非常便宜）。点击右上角的 **「去充值」**。
*建议：普通用户充值 10 元人民币即可使用很久。*

![充值](help/images/deepseek_charge.webp)

**3. 创建 API Key**
点击左侧菜单的 **「API Keys」**，然后点击 **「创建 API Key」** 按钮。
在弹出的窗口中输入名称（例如：`Deadliner`），点击创建。

![创建Key](help/images/deepseek_create.webp)

**4. 复制并保存**
屏幕上会显示一串以 `sk-` 开头的字符，这就是您的密钥。
**注意：** 请立即点击复制并保存好，关闭窗口后将无法再次查看！

## 第二步：在 App 中配置

拿到 Key 之后，回到 Deadliner待办日历 App 进行配置。

**1. 进入设置**
点击底部导航栏最右侧的 **「我的」**，找到并点击 **「AI 智能服务」** 卡片。

**2. 登录华为账号**
如果尚未登录，系统会提示您拉起华为账号登录界面。请授权登录以完成实名校验。

**3. 填写与验证**
在 **API Key** 输入框中，粘贴您刚才复制的 `sk-` 开头的密钥。
点击底部的 **「验证并保存」** 按钮。

系统会发起一次网络请求：
- ✅ **验证成功**：AI 功能即刻开启。
- ❌ **验证失败**：请检查 Key 是否复制完整，或账户余额是否充足。

## 第三步：开始使用

配置成功后，回到主页。

1. 点击主页顶部导航栏的 **AI 助理图标**（星星魔杖图标）。
2. 在弹出的输入框中，粘贴或输入包含任务信息的文本。
3. 点击发送，AI 将自动分析并为您生成任务卡片。
4. 确认信息无误后，点击添加即可。
