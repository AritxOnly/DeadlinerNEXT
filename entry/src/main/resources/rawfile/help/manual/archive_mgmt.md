# 什么是归档管理

归档箱是你的“功勋墙”，也是保持专注的“后台”。它存放了所有你已经完成，或者手动移出的任务与习惯。

通过归档，你可以将主页清空，只关注当下的 Deadline，同时保留随时回顾历史的能力。

## 1. 查看归档

点击主页左上角的 **归档图标** (盒子形状)，即可打开归档面板。

面板顶部提供了切换开关，你可以在这里查看不同类型的历史记录：

- **任务**：按完成时间排序，最近完成的在最上面。卡片上会显示“开始时间 - 完成时间”的时间跨度。
- **习惯**：显示已被归档的习惯。卡片上会显示该习惯的坚持周期（如每日/每周）和最后的目标状态。

## 2. 永久删除

归档箱中的内容是本地存储的。如果你觉得某些记录不再需要，可以将其永久删除。

在归档列表中，每个卡片的右侧都有一个 **红色的垃圾桶按钮**。

1. 点击红色按钮。
2. 系统会弹出确认对话框：“确定要永久删除吗？”。
3. 点击 **删除** 确认。

> **⚠️ 注意**：此操作不可恢复。删除后，该任务将彻底从数据库中消失。

## 常见疑问

**Q: 如何将任务移入归档？**
A: 在主页的任务列表中，通过 <了解手势操作>[gestures] 左滑任务并松手，或者在 <多选模式>[gestures] 下批量操作，即可将任务移入归档箱。

**Q: 完成的任务会自动归档吗？**
A: 是的。当你在主页右滑完成任务后，该任务会被标记为“已完成”并自动进入归档列表，从而保持主页的清爽。
