# 通知与提醒

Deadliner 提供了多维度的提醒机制，确保您不会错过任何一个重要的 Deadline。

## 1. 开启通知权限

要使用提醒功能，首先需要确保 Deadliner 拥有系统通知权限。

在设置页面的 **“通知与提醒”** 中，最上方的开关即为 **总开关**。
- **开启**：如果当前权限被关闭，系统会弹窗请求授权，请点击“允许”。
- **关闭**：由于系统限制，应用无法直接关闭权限。您可以点击该开关，按照提示前往系统的 **设置 -> 通知和状态栏** 中手动关闭。

## 2. 推送类型配置

Deadliner 支持三种类型的智能提醒。您可以根据需要独立控制每一个开关。

### DDL 临近预警 (Smart Alert)
智能检测临近截止日期的任务，并提前发送提醒。
- **触发逻辑**：根据任务的紧急程度（如剩余 1 天、剩余 1 小时）动态调整提醒频率。
- **适用场景**：防止因忙碌而遗忘即将到期的重要作业或报告。

### 每日摘要 (Daily Summary)
每天定时为您推送一份今日待办清单的摘要。
- **配置方法**：
    1. 打开“每日摘要”开关。
    2. 在弹出的时间选择器中设定时间（例如 `08:00` 或 `21:00`）。
    3. 点击确认。
- **适用场景**：晨间规划（早上看要做什么）或 晚间复盘（晚上看还没做什么）。

### 习惯打卡提醒 (Habit Reminder)
针对您设定的长期习惯进行提醒。
- **触发逻辑**：如果到了设定时间您仍未打卡，系统会发送一条友好的提醒。
- **适用场景**：帮助养成“每天背单词”或“健身”的习惯。

## 常见问题

**Q: 为什么通知有时候会迟到？**
A: 目前版本的通知功能基于 **华为代理提醒 API (Reminder Agent)** 实现。
受限于系统对后台应用的省电策略，在某些极端情况下（如手机处于超级省电模式、长时间未打开应用），通知可能会有数分钟的延迟。我们会持续优化这一体验。

**Q: 关闭总开关后，配置会丢失吗？**
A: 不会。您的配置（如每日摘要的时间）会保留在本地。一旦您重新开启系统权限，这些配置将立即恢复生效。
